# Le Petit Prince Leap Game

A browser-based jumping game inspired by *Le Petit Prince*.  
The player leaps on bricks and tries to achieve the highest score possible.

---

## Play the Game

**Live Demo:**  
https://nojinap.github.io/little-prince-game/

---

## About the Project

This project was created as part of a JavaScript assignment to practice building an interactive web application using **HTML, CSS, and JavaScript**.

The goal was to explore:
- DOM manipulation
- Game loops
- Keyboard input
- Collision detection
- Score tracking

---

## Features

- Brick-to-brick jumping mechanics  
- Score tracking system  
- Restart gameplay  
- Simple and clean interface
